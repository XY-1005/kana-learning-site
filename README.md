# kana-learning-site
# 五十音假名学习屋 / Kana Learning Room

一个面向日语零基础学习者的五十音学习网站，包含 **阅读模式** 和 **游戏模式**。  
A Japanese kana learning website for beginners, featuring a **Reading Mode** and a **Game Mode**.

页面采用偏二次元平涂风格的 UI 设计，以浅米色、浅蓝色、棕色为主色调，支持电脑端和手机端自适应展示。  
The UI uses a soft anime-inspired flat design style, with light beige, soft blue, and brown as the main colors. It is responsive on both desktop and mobile devices.

> 这是一个为前端 / Vibe Coding 岗位作品集准备的学习型网页项目，重点展示页面设计、交互逻辑、响应式布局和完整项目交付能力。  
> This is a portfolio-ready learning web project for Frontend / Vibe Coding roles, demonstrating UI design, interaction logic, responsive layout, and complete project delivery.
